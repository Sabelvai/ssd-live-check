## Getting Started

### Prerequisites

* smartmontools
  ```sh
  sudo pacman -S smartmontools
  ```

### Installation

1. Install smartmontools
2. Install Simple SSD Live Check from the Releases tab
3. Make the file executable
   ```sh
   chmod +x ./simplessdstatuscheck
   ```
4. Now let's try to execute it
   ```sh
   ./myscript
   ```
